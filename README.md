# Structured AI Examples

Welcome to the Structured AI Examples repository! This is the central hub for customer-facing integration examples, helper scripts, and automation templates.

## Overview

This repository serves as an **index** to specialized repositories containing production-ready code examples for common integration patterns. Each child repository is focused on a specific use case, making it easy to clone only what you need.

## Available Examples

### 🗄️ [Lakehouse Integration](https://github.com/Structured-AI/structured-ai-lakehouse-integrations)

Export data from Structured AI's Lakehouse using Direct Access and push it to external storage systems.

**Use cases:**

- Export data to AWS S3 buckets
- Sync data to Azure Blob Storage
- Push data to Google Cloud Storage
- Custom ETL pipelines

**What's included:**

- Python scripts with authentication examples
- Configuration templates
- Step-by-step setup guides
- Multiple storage provider examples

## Getting Started

1. **Choose your integration** - Browse the repositories above and select the one that matches your use case
2. **Clone the specific repository** - Each repo is standalone and can be cloned independently
3. **Follow the README** - Each repository contains detailed setup instructions and configuration guides
4. **Customize for your needs** - All examples are designed to be easily modified for your specific requirements

## Repository Structure

This organization follows a **parent-child repository pattern**:

- **structured-ai-examples** (this repo) - Index and documentation hub
- **structured-ai-[integration-name]** - Specialized repositories for each integration type

This structure allows you to:

- ✅ Clone only the code you need
- ✅ Keep examples focused and maintainable
- ✅ Version each integration independently
- ✅ Quickly find relevant examples
